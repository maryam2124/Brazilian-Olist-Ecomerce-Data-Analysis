## Overview
This project contains an **exploratory data analysis (EDA) of the Brazilian Olist E-commerce dataset**. It focuses on analyzing customers, orders, payments, shipping, products, sellers, and reviews to extract actionable insights.

The notebook includes **30 practical data analysis questions** to improve your **Pandas and EDA skills**.

## Dataset
The dataset includes columns such as:

- `customer_id`, `customer_unique_id`, `customer_city`, `customer_state`  
- `order_id`, `order_status`, `order_purchase_timestamp`, `order_delivered_customer_date`  
- `review_score`, `review_comment_message`  
- `payment_type`, `payment_value`, `payment_installments`  
- `product_id`, `product_category_name`, `price`, `freight_value`  
- `seller_id`, `seller_city`, `seller_state`  

> Full dataset: Brazilian Olist E-commerce dataset (customers, orders, payments, products, sellers, reviews)

## Key Analyses
- Missing values identification & handling
